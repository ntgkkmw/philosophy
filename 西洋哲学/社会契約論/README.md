# 社会契約論

## 概要
社会契約論は、国家の正当性を人々の合意に基づいて説明する政治哲学です。自由と秩序の均衡を問います。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 啓蒙思想 | 政治正当化に特化した理論 | 自由と市民性の価値を共有 |
| 功利主義 | 合意起源と結果評価で規範基準が異なる | 制度設計を重視 |
| マルクス主義 | 契約的正当性と階級分析で焦点が異なる | 国家批判の議論点を提供 |
