# トマス・ホッブズ

## 簡易紹介
ホッブズは、自然状態の競争を前提に、社会契約による強力な主権国家の必要性を説いた人物です。

## 生没年・時代背景
- 生没年: 1588-1679
- 時代区分: ルネサンス〜初期近代
- 時代背景: 古典復興と人文主義を背景に、近代科学と政治思想への移行が進んだ時代です。

## 主要著作（代表的な3点）
- 『リヴァイアサン』
- 『法の原理』
- 『市民論』
