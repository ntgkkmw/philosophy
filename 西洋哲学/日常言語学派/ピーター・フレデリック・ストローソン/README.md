# ピーター・フレデリック・ストローソン

## 簡易紹介
ピーター・フレデリック・ストローソンは、日常言語と記述的形而上学を結びつけた分析哲学者です。

## 生没年・時代背景
- 生没年: 1919-2006
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『個体と主語』
- 『意味と真理』
- 『自由と怨恨』

## キーワード（5語）
- 記述的形而上学
- 人格
- 言語分析
- 責任
- 日常概念
