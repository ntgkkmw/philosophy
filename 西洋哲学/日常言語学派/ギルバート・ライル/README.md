# ギルバート・ライル

## 簡易紹介
ギルバート・ライルは、心身二元論を『カテゴリー錯誤』として批判し、日常言語分析を推進した人物です。

## 生没年・時代背景
- 生没年: 1900-1976
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『心の概念』
- 『哲学的議論』
- 『プラトンの進歩』

## キーワード（5語）
