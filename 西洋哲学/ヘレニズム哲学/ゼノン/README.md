# ゼノン

## 簡易紹介
ゼノンはストア派の創始者で、理性に従う生と情念の統御を中心に据えました。

## 生没年・時代背景
- 生没年: c. -334 - c. -262
- 時代区分: 古典期後半〜ヘレニズム
- 時代背景: ポリス秩序の変動と帝国化の中で、世界の理論化と個人の生き方の哲学が同時に深まった時代です。
