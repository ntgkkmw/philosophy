# ヘレニズム哲学

## 概要
ヘレニズム哲学は、古典期後の実践志向の諸学派を含む領域です。個人の生き方と魂の平静を主要テーマにします。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 古代ギリシア哲学 | ポリス中心の政治哲学から個人倫理へ展開 | 徳と幸福の系譜を継承 |
| ストア派 | 広い時代枠と特定学派の関係 | 実践哲学として重なる |
| 懐疑主義 | 確実知への態度が対照的 | 心の平静という目的で接点 |
