# セネカ

## 簡易紹介
セネカは、徳に基づく自己統御と死の受容を説いたローマ・ストア派の哲学者です。

## 生没年・時代背景
- 生没年: c. -4 - 65
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。
