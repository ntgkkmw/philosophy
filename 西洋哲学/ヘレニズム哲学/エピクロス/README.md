# エピクロス

## 簡易紹介
エピクロスは、快楽を苦痛の不在として捉え、節度ある生活と心の平静を最高善とした哲学者です。

## 生没年・時代背景
- 生没年: -341 - -270
- 時代区分: 古典期後半〜ヘレニズム
- 時代背景: ポリス秩序の変動と帝国化の中で、世界の理論化と個人の生き方の哲学が同時に深まった時代です。
