# ラッセル

## 簡易紹介
ラッセルは、論理分析によって哲学問題を解く方針を示し、記述理論などで分析哲学を牽引した人物です。

## 生没年・時代背景
- 生没年: 1872-1970
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『数学原理』
- 『哲学の諸問題』
- 『確定記述の理論 論文』

## キーワード（5語）
- 論理分析
- 記述理論
- 論理主義
- 外在的関係
- 明晰性

## 読書導線（入門→中核→発展）
1. 入門: 『数学原理』
2. 中核: 『哲学の諸問題』
3. 発展: 『確定記述の理論 論文』
