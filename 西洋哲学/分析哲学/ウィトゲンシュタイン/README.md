# ウィトゲンシュタイン

## 簡易紹介
ウィトゲンシュタインは、前期に論理像理論、後期に言語ゲーム論を提示し、言語観を大きく転換させた人物です。

## 生没年・時代背景
- 生没年: 1889-1951
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『論理哲学論考』
- 『哲学探究』
- 『確実性について』

## キーワード（5語）
- 言語ゲーム
- 家族的類似
- 論理像理論
- 規則遵守
- 沈黙
