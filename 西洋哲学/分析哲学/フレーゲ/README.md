# フレーゲ

## 簡易紹介
フレーゲは、近代記号論理学を創始し、意味と指示の区別で分析哲学の基礎を築いた人物です。

## 生没年・時代背景
- 生没年: 1848-1925
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『概念記法』
- 『算術の基礎』
- 『意味と意義について』

## キーワード（5語）
- 論理主義
- 意味と指示
- 関数引数
- 反心理主義
- 分析哲学
