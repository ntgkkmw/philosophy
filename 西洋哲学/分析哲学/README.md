# 分析哲学

## 概要
分析哲学は、論理と言語分析を中核手法とし、議論の明晰さと概念の厳密化を重視する現代哲学の潮流です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 論理実証主義 | 学派内でも検証主義への距離がある | 明晰さと論理を共有 |
| 日常言語学派 | 形式言語重視と日常使用重視で方法が分かれる | 概念混乱の解消を目指す |
| 現象学 | 第三者的分析と第一人称記述でアプローチが異なる | 心と言語への関心を共有 |
