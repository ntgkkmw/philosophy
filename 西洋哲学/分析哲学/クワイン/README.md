# クワイン

## 簡易紹介
クワインは、分析/総合の区別を批判し、自然主義的な知識観へ分析哲学を方向づけた人物です。

## 生没年・時代背景
- 生没年: 1908-2000
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『経験主義の二つのドグマ』
- 『ことばと対象』
- 『存在論的相対性について』

## キーワード（5語）
- 自然主義
- 存在論的コミットメント
- 全体論
- 翻訳不確定性
- 経験主義批判
