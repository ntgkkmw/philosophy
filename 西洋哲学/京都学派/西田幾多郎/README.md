# 西田幾多郎

## 簡易紹介
西田幾多郎は、純粋経験と場所の論理を軸に西洋哲学と東洋思想を接続した京都学派の中心人物です。

## 生没年・時代背景
- 生没年: 1870-1945
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『善の研究』
- 『自覚に於ける直観と反省』
- 『場所的論理と宗教的世界観』
