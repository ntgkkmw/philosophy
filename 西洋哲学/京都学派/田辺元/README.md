# 田辺元

## 簡易紹介
田辺元は、種の論理や懺悔道を通じて実存と歴史の課題に応答した京都学派の哲学者です。

## 生没年・時代背景
- 生没年: 1885-1962
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。
