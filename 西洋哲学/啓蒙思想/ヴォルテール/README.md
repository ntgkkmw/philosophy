# ヴォルテール

## 簡易紹介
ヴォルテールは、宗教的不寛容や権威主義を批判し、理性と寛容を擁護した啓蒙思想家です。

## 生没年・時代背景
- 生没年: 1694-1778
- 時代区分: 17世紀近世
- 時代背景: 科学革命と方法論の革新を背景に、理性と経験の基礎づけを巡る論争が加速した時代です。

## 主要著作（代表的な3点）

- 『寛容論』
