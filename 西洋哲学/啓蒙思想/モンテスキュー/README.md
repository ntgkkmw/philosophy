# モンテスキュー

## 簡易紹介
モンテスキューは、権力分立の理論を通じて専制の抑制と法の支配を構想した政治思想家です。

## 生没年・時代背景
- 生没年: 1689-1755
- 時代区分: 17世紀近世
- 時代背景: 科学革命と方法論の革新を背景に、理性と経験の基礎づけを巡る論争が加速した時代です。

## 主要著作（代表的な3点）
- 『法の精神』
- 『ペルシア人の手紙』
- 『ローマ盛衰原因論』
