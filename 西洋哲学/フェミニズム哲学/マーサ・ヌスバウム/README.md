# マーサ・ヌスバウム

## 簡易紹介
マーサ・ヌスバウムは、人間の尊厳を実質的能力で評価するケイパビリティ・アプローチを発展させた人物です。

## 生没年・時代背景
- 生没年: 1947-
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『善の脆さ』
- 『詩的正義』
- 『人間性の能力アプローチ』

## キーワード（5語）

- 実践理性
