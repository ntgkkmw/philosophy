# ジュディス・バトラー

## 簡易紹介
ジュディス・バトラーは、ジェンダーを反復的実践として捉えるパフォーマティヴィティ理論を提示した人物です。

## 生没年・時代背景
- 生没年: 1956-
- 時代区分: 20世紀後半〜現代
- 時代背景: 脱構築・ジェンダー・科学哲学などが展開し、多元的で学際的な哲学実践が広がった時代です。

## 主要著作（代表的な3点）

- 『問題=物質となる身体』
- 『触発する言葉』
