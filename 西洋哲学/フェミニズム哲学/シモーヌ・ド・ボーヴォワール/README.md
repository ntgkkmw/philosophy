# シモーヌ・ド・ボーヴォワール

## 簡易紹介
シモーヌ・ド・ボーヴォワールは、女性が社会的に『第二の性』として構成される過程を批判した思想家です。

## 生没年・時代背景
- 生没年: 1908-1986
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。
