# 新プラトン主義

## 概要
新プラトン主義は、プラトン哲学を再編して『一者』からの流出という形而上学を構築した後期古代の思想です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 古代ギリシア哲学 | プラトン継承を形而上学化 | 超越的原理探究を共有 |
| 中世スコラ哲学 | 神学的再編の程度が異なる | 存在階梯の発想を共有 |
| 現象学 | 流出論と記述的方法で相違 | 意識と超越の問題で接点 |
