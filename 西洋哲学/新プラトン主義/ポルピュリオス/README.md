# ポルピュリオス

## 簡易紹介
ポルピュリオスは、プロティノス思想の継承者であり、論理学と形而上学の橋渡しを担った人物です。

## 生没年・時代背景
- 生没年: c. 234-305
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。

## 主要著作（代表的な3点）
- 『エイサゴーゲー』
- 『プロティノス伝』
- 『キリスト教徒駁論 断片』
