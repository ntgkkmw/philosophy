# プロティノス

## 簡易紹介
プロティノスは、『一者』から知性と魂が流出するという新プラトン主義の体系を築いた人物です。

## 生没年・時代背景
- 生没年: 204-270
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。
