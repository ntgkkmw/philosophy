# J・S・ミル

## 簡易紹介
J・S・ミルは、功利主義に質的区別を導入し、個人の自由と社会的幸福の両立を論じた人物です。

## 生没年・時代背景
- 生没年: 1806-1873
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『自由論』
- 『功利主義』
- 『代議制統治論』
