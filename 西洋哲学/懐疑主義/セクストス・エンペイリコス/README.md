# セクストス・エンペイリコス

## 簡易紹介
セクストス・エンペイリコスは、懐疑主義の方法を体系化し、判断停止の実践を詳述した人物です。

## 生没年・時代背景
- 生没年: c. 160 - c. 210
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。
