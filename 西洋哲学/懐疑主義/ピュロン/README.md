# ピュロン

## 簡易紹介
ピュロンは、断定を保留することで心の平静に至るという懐疑主義の基礎を築いた人物です。

## 生没年・時代背景
- 生没年: c. -360 - c. -270
- 時代区分: 古典期後半〜ヘレニズム
- 時代背景: ポリス秩序の変動と帝国化の中で、世界の理論化と個人の生き方の哲学が同時に深まった時代です。

## 主要著作（代表的な3点）
- 『断片集』
- 『ピュロン主義資料集』
