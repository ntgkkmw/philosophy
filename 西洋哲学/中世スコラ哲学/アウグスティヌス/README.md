# アウグスティヌス

## 簡易紹介
アウグスティヌスは、内面的自己探究と神の恩寵を結び付け、中世キリスト教思想の基盤を作った人物です。

## 生没年・時代背景
- 生没年: 354-430
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。
