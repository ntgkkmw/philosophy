# トマス・アクィナス

## 簡易紹介
トマス・アクィナスは、理性と信仰の調和を目指し、神学とアリストテレス哲学を統合した人物です。

## 生没年・時代背景
- 生没年: 1225-1274
- 時代区分: 中世
- 時代背景: 神学と哲学の統合が主題となり、論理と体系化の技法が大きく発達した時代です。

## 主要著作（代表的な3点）
- 『神学大全』
- 『異教徒大全』
- 『存在者と本質について』

## キーワード（5語）
