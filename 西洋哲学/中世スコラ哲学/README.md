# 中世スコラ哲学

## 概要
中世スコラ哲学は、キリスト教神学とアリストテレス哲学を精密な論証で統合しようとした知的運動です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 古代ギリシア哲学 | 啓示と神学が理論枠を与える | 論理学と形而上学を継承 |
| ルネサンス哲学 | 権威中心と人文主義中心で差がある | 古典継承の問題意識は共通 |
| 合理論 | 神学的前提の強さが異なる | 体系構築志向を共有 |
