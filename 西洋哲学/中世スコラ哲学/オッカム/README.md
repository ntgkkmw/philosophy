# オッカム

## 簡易紹介
オッカムは、普遍を実在視しない唯名論を推し進め、『オッカムの剃刀』で知られる人物です。

## 生没年・時代背景
- 生没年: c. 1287-1347
- 時代区分: 中世
- 時代背景: 神学と哲学の統合が主題となり、論理と体系化の技法が大きく発達した時代です。

## 主要著作（代表的な3点）
- 『論理学大全』
- 『命題集注解』
- 『神学大全注解』
