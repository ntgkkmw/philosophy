# 経験論

## 概要
経験論は、知識の起点を感覚経験に置く立場です。観察と経験を重視し、近代認識論を大きく前進させました。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 合理論 | 感覚経験起点と理性原理起点で対照 | 知識の確実性を追求 |
| 分析哲学 | 心理学的経験論から言語論理分析へ展開 | 明晰化志向を共有 |
| プラグマティズム | 受動的知覚分析と実践的真理観で重心が異なる | 経験重視を共有 |
