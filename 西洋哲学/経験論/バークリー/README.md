# バークリー

## 簡易紹介
バークリーは、『存在するとは知覚されることである』とする観念論を展開した人物です。

## 生没年・時代背景
- 生没年: 1685-1753
- 時代区分: 17世紀近世
- 時代背景: 科学革命と方法論の革新を背景に、理性と経験の基礎づけを巡る論争が加速した時代です。

## 主要著作（代表的な3点）
- 『人知原理論』
- 『ハイラスとフィロナスの三つの対話』
- 『視覚新論』
