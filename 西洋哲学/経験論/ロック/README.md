# ロック

## 簡易紹介
ロックは、心を白紙とみなす経験論と、権利を基礎にした自由主義政治思想を打ち立てた人物です。

## 生没年・時代背景
- 生没年: 1632-1704
- 時代区分: 17世紀近世
- 時代背景: 科学革命と方法論の革新を背景に、理性と経験の基礎づけを巡る論争が加速した時代です。
