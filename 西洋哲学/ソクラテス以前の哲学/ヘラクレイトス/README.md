# ヘラクレイトス

## 簡易紹介
ヘラクレイトスは、万物流転とロゴスの思想で、世界を対立と変化の秩序として捉えた人物です。

## 生没年・時代背景
- 生没年: c. -540 - c. -480
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
