# タレス

## 簡易紹介
タレスは、万物の根源を水と捉え、神話ではなく自然的説明で世界を理解しようとした初期哲学者です。

## 生没年・時代背景
- 生没年: c. -624 - c. -546
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
