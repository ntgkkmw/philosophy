# エンペドクレス

## 簡易紹介
エンペドクレスは、四元素と愛・憎の力によって生成変化を説明した多元論の哲学者です。

## 生没年・時代背景
- 生没年: c. -494 - c. -434
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。

## 主要著作（代表的な3点）
- 『自然について』
- 『浄めについて』
- 『ソクラテス以前哲学者断片集』

## キーワード（5語）
