# アナクサゴラス

## 簡易紹介
アナクサゴラスは、宇宙秩序の原理としてヌース（知性）を導入した人物です。

## 生没年・時代背景
- 生没年: c. -500 - c. -428
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。

- 『ソクラテス以前哲学者断片集』
- 『ギリシア哲学者列伝』
