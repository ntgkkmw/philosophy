# アナクシメネス

## 簡易紹介
アナクシメネスは、万物の根源を空気とし、希薄化と凝縮で世界の変化を説明した人物です。

## 生没年・時代背景
- 生没年: c. -586 - c. -526
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
