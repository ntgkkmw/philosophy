# パルメニデス

## 簡易紹介
パルメニデスは、真に在るものは不生不滅で不変だと主張し、感覚世界より理性を優先した人物です。

## 生没年・時代背景
- 生没年: c. -515 - c. -450
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
