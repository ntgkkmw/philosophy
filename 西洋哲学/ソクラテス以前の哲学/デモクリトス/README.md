# デモクリトス

## 簡易紹介
デモクリトスは、不可分な原子と空虚から世界を説明した原子論の代表者です。

## 生没年・時代背景
- 生没年: c. -460 - c. -370
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。

## 主要著作（代表的な3点）
- 『断片集』
- 『ソクラテス以前哲学者断片集』
- アリストテレス『自然学』言及
