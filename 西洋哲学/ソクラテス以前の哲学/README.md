# ソクラテス以前の哲学

## 概要
ソクラテス以前の哲学は、世界の根源（アルケー）を神話から切り離して説明しようとした思索です。自然哲学の原型を与えました。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 古代ギリシア哲学 | 形而上学体系化以前の自然哲学が中心 | ロゴス志向を共有 |
| ヘレニズム哲学 | 宇宙論中心と人生術中心で焦点が異なる | 自然理解と人間理解が接続 |
| 合理論 | 経験的観察の比重が高い | 第一原理志向という接点 |
