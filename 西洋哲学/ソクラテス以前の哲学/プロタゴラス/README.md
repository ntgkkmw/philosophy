# プロタゴラス

## 簡易紹介
プロタゴラスは、『人間は万物の尺度である』と述べ、相対主義的認識論を示したソフィストです。

## 生没年・時代背景
- 生没年: c. -490 - c. -420
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
