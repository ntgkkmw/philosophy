# ゴルギアス

## 簡易紹介
ゴルギアスは、存在や認識の確実性に懐疑的立場を取り、言語と修辞の力を前面に出したソフィストです。

## 生没年・時代背景
- 生没年: c. -483 - c. -375
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
