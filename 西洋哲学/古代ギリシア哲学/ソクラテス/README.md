# ソクラテス

## 簡易紹介
ソクラテスは、対話による吟味を通じて『善く生きること』を問い、無知の自覚を哲学の出発点に据えた人物です。

## 生没年・時代背景
- 生没年: c. -470 - -399
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。

## 主要著作（代表的な3点）
- プラトン『ソクラテスの弁明』
- プラトン『クリトン』
- クセノポン『ソクラテスの思い出』
