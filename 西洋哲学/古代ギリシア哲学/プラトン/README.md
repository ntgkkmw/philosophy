# プラトン

## 簡易紹介
プラトンは、イデア論を軸に認識・倫理・政治を体系化し、哲学を学問として定着させた人物です。

## 生没年・時代背景
- 生没年: c. -427 - c. -347
- 時代区分: 古代ギリシア（前ソクラテス〜古典期）
- 時代背景: ポリスの形成と神話からロゴスへの転換が進み、自然・存在・倫理の基本語彙が成立した時代です。
