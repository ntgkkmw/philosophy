# アリストテレス

## 簡易紹介
アリストテレスは、論理学・形而上学・倫理学を体系化し、四原因説と徳倫理で大きな影響を与えた人物です。

## 生没年・時代背景
- 生没年: -384 - -322
- 時代区分: 古典期後半〜ヘレニズム
- 時代背景: ポリス秩序の変動と帝国化の中で、世界の理論化と個人の生き方の哲学が同時に深まった時代です。

## 主要著作（代表的な3点）
- 『ニコマコス倫理学』
- 『形而上学』
- 『オルガノン』

## キーワード（5語）
