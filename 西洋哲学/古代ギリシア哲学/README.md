# 古代ギリシア哲学

## 概要
古代ギリシア哲学は、西洋哲学の基盤を築いた時代です。存在論・認識論・倫理学・政治哲学の基本語彙が形成されました。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| ソクラテス以前の哲学 | 自然の始原探究から倫理政治へ重心が移る | 理性による世界理解を共有 |
| ヘレニズム哲学 | 共同体理想から個人の生き方へ重点が移る | 徳と幸福の探究を継承 |
| 中世スコラ哲学 | 神学統合の有無で方法が異なる | 論証重視の態度は継承 |
