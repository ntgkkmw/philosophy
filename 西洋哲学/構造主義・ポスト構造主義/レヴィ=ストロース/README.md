# レヴィ=ストロース

## 簡易紹介
レヴィ=ストロースは、神話や親族構造の分析から人間文化の深層構造を明らかにした思想家です。

## 生没年・時代背景
- 生没年: 1908-2009
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。
