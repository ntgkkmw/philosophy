# フーコー

## 簡易紹介
フーコーは、知と権力の関係を歴史的に分析し、主体形成のメカニズムを解明した人物です。

## 生没年・時代背景
- 生没年: 1926-1984
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『言葉と物』
- 『監獄の誕生』
- 『性の歴史1 知への意志』
