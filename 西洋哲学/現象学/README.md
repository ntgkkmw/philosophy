# 現象学

## 概要
現象学は、先入見を括弧に入れ、意識に現れる経験の構造を記述する方法です。20世紀大陸哲学の基盤を作りました。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 実存主義 | 記述方法論と実存的問いで重心が異なる | 主体経験を重視 |
| 解釈学 | 純粋記述と解釈の不可避性で差がある | 理解の条件を問う |
| 分析哲学 | 第一人称記述と論理分析で方法が異なる | 意識と言語の問題で接点 |
