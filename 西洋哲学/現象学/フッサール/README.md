# フッサール

## 簡易紹介
フッサールは、意識の志向性とエポケーを軸に現象学を創始した人物です。

## 生没年・時代背景
- 生没年: 1859-1938
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）

- 『イデーンI』
- 『デカルト的省察』
