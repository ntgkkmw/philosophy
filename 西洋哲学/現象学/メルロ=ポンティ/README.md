# メルロ=ポンティ

## 簡易紹介
メルロ=ポンティは、身体を通じた知覚経験を重視し、現象学を存在論へ拡張した人物です。

## 生没年・時代背景
- 生没年: 1908-1961
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『知覚の現象学』
- 『行動の構造』
- 『見えるものと見えないもの』

## キーワード（5語）
