# ルクレティウス

## 簡易紹介
ルクレティウスは、エピクロス思想を詩的に展開し、自然理解による恐怖の克服を説いた人物です。

## 生没年・時代背景
- 生没年: c. -99 - c. -55
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。

## 主要著作（代表的な3点）
