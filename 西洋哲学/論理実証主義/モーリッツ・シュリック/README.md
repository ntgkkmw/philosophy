# モーリッツ・シュリック

## 簡易紹介
モーリッツ・シュリックは、ウィーン学団を主導し、検証可能性を基準とする哲学観を推進した人物です。

## 生没年・時代背景
- 生没年: 1882-1936
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『一般認識論』
- 『倫理の諸問題』
- 『意味と検証』

## キーワード（5語）
- 検証
- 反形而上学
- 科学的世界観
- 経験
- 論理分析
