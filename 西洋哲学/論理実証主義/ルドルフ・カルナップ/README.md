# ルドルフ・カルナップ

## 簡易紹介
ルドルフ・カルナップは、論理実証主義を体系化し、形而上学的命題の意味分析を徹底した人物です。

## 生没年・時代背景
- 生没年: 1891-1970
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『世界の論理的構築』
- 『言語の論理的構文論』
- 『形而上学の克服』

## キーワード（5語）
- 論理実証主義
- 構文論
- 寛容原理
- 検証
- 科学統一
