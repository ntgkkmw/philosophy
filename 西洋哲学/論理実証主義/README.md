# 論理実証主義

## 概要
論理実証主義は、経験的検証可能性を意味の基準に据え、形而上学的命題の整理・排除を目指した運動です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 分析哲学 | 検証可能性をより強く要求 | 論理分析を共有 |
| 日常言語学派 | 人工言語志向と日常言語志向で対照 | 意味の明確化を共有 |
| 現象学 | 主観記述を形而上学とみなしがち | 科学的方法の議論で接点 |
