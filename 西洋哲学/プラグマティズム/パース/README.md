# パース

## 簡易紹介
パースは、概念の意味を実践的帰結で測るプラグマティズムを提唱した人物です。

## 生没年・時代背景
- 生没年: 1839-1914
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。
