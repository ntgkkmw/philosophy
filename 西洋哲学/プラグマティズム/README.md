# プラグマティズム

## 概要
プラグマティズムは、概念や理論の意味を実践的帰結から評価する立場です。知識を固定的真理ではなく探究過程として捉えます。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 経験論 | 受動的経験分析より問題解決実践を重視 | 経験に根ざす姿勢を共有 |
| 分析哲学 | 形式厳密性より実践的有効性を重視 | 言語と意味の検討で接点 |
| 実存主義 | 個の決断より共同的探究を前景化 | 反本質主義で接続 |
