# デューイ

## 簡易紹介
デューイは、探究を社会的実践として捉え、教育哲学と民主主義論に大きな影響を与えた人物です。

## 生没年・時代背景
- 生没年: 1859-1952
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
