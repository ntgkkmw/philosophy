# ウィリアム・ジェームズ

## 簡易紹介
ウィリアム・ジェームズは、真理を生の実践に根ざして捉え、経験の多様性を重視したプラグマティストです。

## 生没年・時代背景
- 生没年: 1842-1910
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『心理学原理』
- 『プラグマティズム』
- 『宗教的経験の諸相』

## キーワード（5語）
- 経験
- 真理
- 有用性
- 意識の流れ
- 多元論
