# 西洋哲学ディレクトリ運用ガイド

このディレクトリは、西洋哲学を「時代」「流派」「代表人物」で体系的に深掘りするためのベースです。

## 目的
- 学習と執筆の導線を固定化する
- 流派ごとの論点比較をしやすくする
- 人物ごとのメモを再利用できる形で蓄積する

## 基本構造
- 第1階層: 流派・思想運動・時代カテゴリ
- 第2階層: 代表的な哲学者

例:
- `西洋哲学/古代ギリシア哲学/プラトン`
- `西洋哲学/分析哲学/ウィトゲンシュタイン`
- `西洋哲学/批判理論/テオドール・アドルノ`

## 学習導線（推奨）
1. 古代
   - `古代ギリシア哲学`
   - `ソクラテス以前の哲学`
   - `ヘレニズム哲学`
   - `ストア派`
   - `エピクロス派`
2. 中世〜ルネサンス
   - `中世スコラ哲学`
   - `新プラトン主義`
   - `ルネサンス哲学`
3. 近世
   - `経験論`
   - `合理論`
   - `啓蒙思想`
   - `社会契約論`
4. 19世紀
   - `ドイツ観念論`
   - `イギリス観念論`
   - `功利主義`
   - `マルクス主義`
   - `プラグマティズム`
5. 20世紀以降
   - `分析哲学`
   - `論理実証主義`
   - `日常言語学派`
   - `現象学`
   - `解釈学`
   - `実存主義`
   - `批判理論`
   - `構造主義・ポスト構造主義`
   - `ポストモダニズム`
   - `フェミニズム哲学`

## Wikipedia連動データ
- `西洋哲学/Wikipediaリンク一覧.tsv`
  - 列: `section`, `title`, `url`
  - Wikipedia「西洋哲学」ページの見出し別リンク索引
- `西洋哲学/Wikipediaリンク集.md`
  - 見出しごとにクリック可能なリンクを一覧化

## 追加運用ルール
- 新しい流派を追加する時は、第1階層に流派名フォルダを作る
- 人物を追加する時は、対応する流派配下に人物フォルダを作る
- 人物フォルダには将来的に `README.md` を置き、以下を統一項目にする
  - 生没年
  - 主要著作
  - 中核概念
  - 他流派への影響
