# 実存主義

## 概要
実存主義は、人間が先に世界へ投げ出され、選択と責任を引き受けながら自己を形成するという立場です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 現象学 | 方法論基盤を受けつつ価値選択へ拡張 | 主体の具体的生を重視 |
| ストア派 | 宇宙秩序受容と不条理世界認識で差異 | 自己形成と実践倫理で接点 |
| ポストモダニズム | 主体の一貫性評価で立場が異なる | 近代主体批判で交差 |
