# キルケゴール

## 簡易紹介
キルケゴールは、主体的真理と信仰の跳躍を説き、実存思想の先駆けとなった人物です。

## 生没年・時代背景
- 生没年: 1813-1855
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『あれか これか』
- 『死に至る病』
- 『恐れとおののき』

## キーワード（5語）
- 実存
- 主体性
- 不安
- 絶望
- 信仰
