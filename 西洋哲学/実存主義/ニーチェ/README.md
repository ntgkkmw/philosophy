# ニーチェ

## 簡易紹介
ニーチェは、既存道徳の価値転換を促し、力への意志と生の肯定を打ち出した人物です。

## 生没年・時代背景
- 生没年: 1844-1900
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『善悪の彼岸』
- 『ツァラトゥストラはかく語りき』
- 『道徳の系譜』
