# サルトル

## 簡易紹介
サルトルは、『実存は本質に先立つ』と述べ、自由と責任を中心に実存主義を展開した人物です。

## 生没年・時代背景
- 生没年: 1905-1980
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『存在と無』
- 『実存主義とは何か』
- 『弁証法的理性批判』
