# ハイデガー

## 簡易紹介
ハイデガーは、存在の問いを再起動し、現存在の時間性と世界内存在を分析した人物です。

## 生没年・時代背景
- 生没年: 1889-1976
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）

- 『形而上学入門』
- 『技術への問い』
