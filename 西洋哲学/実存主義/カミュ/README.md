# カミュ

## 簡易紹介
カミュは、世界の不条理を直視しつつ、反抗と連帯によって生を引き受ける姿勢を示した人物です。

## 生没年・時代背景
- 生没年: 1913-1960
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『シーシュポスの神話』
- 『異邦人』
- 『反抗的人間』

## キーワード（5語）

- 限界
