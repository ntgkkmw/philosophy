# ライプニッツ

## 簡易紹介
ライプニッツは、モナド論と充足理由律を軸に、合理的な世界秩序を構想した哲学者です。

## 生没年・時代背景
- 生没年: 1646-1716
- 時代区分: 17世紀近世
- 時代背景: 科学革命と方法論の革新を背景に、理性と経験の基礎づけを巡る論争が加速した時代です。

## 主要著作（代表的な3点）
- 『モナドロジー』
- 『形而上学叙説』
- 『人間知性新論』

## キーワード（5語）

- 合理性
