# デカルト

## 簡易紹介
デカルトは、方法的懐疑から『我思う、ゆえに我あり』に到達し、近代哲学の出発点を作った人物です。

## 生没年・時代背景
- 生没年: 1596-1650
- 時代区分: ルネサンス〜初期近代
- 時代背景: 古典復興と人文主義を背景に、近代科学と政治思想への移行が進んだ時代です。

## 主要著作（代表的な3点）
- 『方法序説』
- 『省察』
- 『哲学原理』

## キーワード（5語）
- 方法的懐疑
- コギト
- 二元論
- 明証性
- 演繹

## 読書導線（入門→中核→発展）
1. 入門: 『方法序説』
2. 中核: 『省察』
3. 発展: 『哲学原理』
