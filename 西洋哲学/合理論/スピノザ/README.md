# スピノザ

## 簡易紹介
スピノザは、神と自然を同一視する一元論を展開し、必然性の中で自由を再定義した人物です。

## 生没年・時代背景
- 生没年: 1632-1677
- 時代区分: 17世紀近世
- 時代背景: 科学革命と方法論の革新を背景に、理性と経験の基礎づけを巡る論争が加速した時代です。

## 主要著作（代表的な3点）

- 『知性改善論』
