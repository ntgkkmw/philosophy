# 合理論

## 概要
合理論は、理性と演繹を知識の中核に置く立場です。確実性の高い原理から体系を組み立てることを目指します。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 経験論 | 生得観念と感覚経験で根拠が異なる | 方法論的厳密さを追求 |
| ドイツ観念論 | 実体論中心から主体構成論へ展開 | 体系志向を継承 |
| 論理実証主義 | 形而上学肯定と反形而上学で立場が異なる | 論理の重視で接点 |
