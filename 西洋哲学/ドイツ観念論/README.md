# ドイツ観念論

## 概要
ドイツ観念論は、カント以後に展開した意識・自我・歴史の体系哲学です。主体と世界の関係を根本から再構成しました。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較
