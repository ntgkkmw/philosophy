# マルクス主義

## 概要
マルクス主義は、歴史・経済・社会を階級関係から分析し、資本主義批判と社会変革を目指す思想です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| ドイツ観念論 | 観念の自己運動から物質的歴史分析へ転換 | 弁証法を継承 |
| 批判理論 | 経済決定論傾向と文化権力分析の違い | 資本主義批判を共有 |
| 功利主義 | 分配正義の基礎づけが異なる | 社会制度改革志向で接点 |
