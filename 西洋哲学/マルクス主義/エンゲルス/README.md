# エンゲルス

## 簡易紹介
エンゲルスは、マルクスと協働して歴史唯物論を体系化し、社会主義理論の普及に貢献した人物です。

## 生没年・時代背景
- 生没年: 1820-1895
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『反デューリング論』
- 『家族 私有財産 国家の起源』
- 『自然弁証法』
