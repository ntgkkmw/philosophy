# カール・マルクス

## 簡易紹介
カール・マルクスは、歴史唯物論によって資本主義の構造を分析し、階級闘争を歴史変動の軸に据えた人物です。

## 生没年・時代背景
- 生没年: 1818-1883
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）

- 『経済学 哲学草稿』
- 『ドイツ イデオロギー』
