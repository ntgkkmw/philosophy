# フランシス・ベーコン

## 簡易紹介
フランシス・ベーコンは、経験と帰納を重視する新しい学問方法を提唱した近代科学思想の先駆者です。

## 生没年・時代背景
- 生没年: 1561-1626
- 時代区分: ルネサンス〜初期近代
- 時代背景: 古典復興と人文主義を背景に、近代科学と政治思想への移行が進んだ時代です。
