# トマス・モア

## 簡易紹介
トマス・モアは、理想社会像『ユートピア』を通じて当時の政治・社会を批判的に描いた人物です。

## 生没年・時代背景
- 生没年: 1478-1535
- 時代区分: ルネサンス〜初期近代
- 時代背景: 古典復興と人文主義を背景に、近代科学と政治思想への移行が進んだ時代です。

## 主要著作（代表的な3点）
- 『ユートピア』
- 『リチャード三世史』
- 『対話集』
