# マルシリオ・フィチーノ

## 簡易紹介
マルシリオ・フィチーノは、プラトン哲学の復興を通じてルネサンス人文主義に影響を与えた人物です。

## 生没年・時代背景
- 生没年: 1433-1499
- 時代区分: ルネサンス〜初期近代
- 時代背景: 古典復興と人文主義を背景に、近代科学と政治思想への移行が進んだ時代です。

## 主要著作（代表的な3点）
- 『プラトン神学』
- 『プラトン全集ラテン語訳』
- 『魂の不死について』
