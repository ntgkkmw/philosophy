# ニッコロ・マキャヴェッリ

## 簡易紹介
ニッコロ・マキャヴェッリは、現実の権力作用に即した政治分析を行い、近代政治思想の基礎を築いた人物です。

## 生没年・時代背景
- 生没年: 1469-1527
- 時代区分: ルネサンス〜初期近代
- 時代背景: 古典復興と人文主義を背景に、近代科学と政治思想への移行が進んだ時代です。

- 『フィレンツェ史』
