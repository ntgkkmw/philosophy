# 解釈学

## 概要
解釈学は、テクスト・歴史・対話の理解条件を問う学問です。理解は常に歴史性と前理解を伴うと考えます。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 現象学 | 記述中心と歴史的解釈中心で重点が異なる | 理解経験を重視 |
| 日常言語学派 | 文脈歴史性と語用規則の焦点が異なる | 意味の成立条件を問う |
| ポストモダニズム | 伝統との対話と脱中心化で差がある | テクスト解釈で接点 |
