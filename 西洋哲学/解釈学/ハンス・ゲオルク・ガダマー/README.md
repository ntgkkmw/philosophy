# ハンス・ゲオルク・ガダマー

## 簡易紹介
ガダマーは、理解を歴史的対話過程として捉え、『地平融合』の概念を示した解釈学者です。

## 生没年・時代背景
- 生没年: 1900-2002
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『真理と方法』
- 『哲学的解釈学』
- 『理性の時代』
