# ジャック・デリダ

## 簡易紹介
デリダは、テクスト内部の差延と二項対立の不安定性を示す脱構築を展開した人物です。

## 生没年・時代背景
- 生没年: 1930-2004
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。
