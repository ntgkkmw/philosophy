# ジル・ドゥルーズ

## 簡易紹介
ドゥルーズは、差異・生成・多様体の概念を軸に固定的同一性を批判した哲学者です。

## 生没年・時代背景
- 生没年: 1925-1995
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。

## 主要著作（代表的な3点）
- 『差異と反復』
- 『意味の論理学』
- 『千のプラトー』

## キーワード（5語）
