# ポストモダニズム

## 概要
ポストモダニズムは、普遍的真理や単一の進歩史観を相対化し、知と権力の関係を批判的に検討する潮流です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 構造主義・ポスト構造主義 | 構造把握より差異と断片性を強調 | 大きな物語批判を共有 |
| 批判理論 | 規範理論の扱いで立場が分かれる | 権力批判を共有 |
| フェミニズム哲学 | 主体概念の扱いで緊張がある | 同一性批判で接点 |
