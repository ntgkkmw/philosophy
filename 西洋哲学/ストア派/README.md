# ストア派

## 概要
ストア派は、理性に従って生きる徳と内面的平静を重視する実践哲学です。制御可能なものに集中する態度を説きます。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| エピクロス派 | 徳中心と快苦調整中心で倫理軸が違う | 平静の追求を共有 |
| 実存主義 | 宇宙秩序への同意と不条理への応答で前提が異なる | 自己責任の強調は共通 |
| 功利主義 | 内面的徳を優先し結果計算を相対化 | 実践的倫理として現代応用可能 |
