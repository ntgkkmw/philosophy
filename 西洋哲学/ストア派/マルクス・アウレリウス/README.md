# マルクス・アウレリウス

## 簡易紹介
マルクス・アウレリウスは、日々の省察を通じて理性と義務に生きる姿勢を示した皇帝哲学者です。

## 生没年・時代背景
- 生没年: 121-180
- 時代区分: ローマ帝政期〜古代末期
- 時代背景: 帝国秩序と宗教的世界観の交錯の中で、実践倫理と超越的秩序の探究が重なった時代です。
