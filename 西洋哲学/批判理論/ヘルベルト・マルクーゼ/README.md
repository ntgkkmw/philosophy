# ヘルベルト・マルクーゼ

## 簡易紹介
マルクーゼは、先進産業社会の同調圧力を批判し、解放の可能性を探った批判理論家です。

## 生没年・時代背景
- 生没年: 1898-1979
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『エロスと文明』
- 『一次元的人間』
- 『理性と革命』
