# テオドール・アドルノ

## 簡易紹介
アドルノは、啓蒙理性の自己破壊を批判し、否定弁証法によって社会批判を深めた人物です。

## 生没年・時代背景
- 生没年: 1903-1969
- 時代区分: 20世紀前半
- 時代背景: 世界大戦と社会変動を背景に、言語・存在・社会批判をめぐる新しい哲学潮流が形成された時代です。
