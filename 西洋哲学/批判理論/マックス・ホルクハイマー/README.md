# マックス・ホルクハイマー

## 簡易紹介
ホルクハイマーは、伝統理論と批判理論を区別し、社会変革志向の理論を提唱した人物です。

## 生没年・時代背景
- 生没年: 1895-1973
- 時代区分: 19世紀
- 時代背景: 産業化と国民国家の時代に、歴史・社会・主体を捉える大規模理論が展開した時代です。

## 主要著作（代表的な3点）
- 『伝統理論と批判理論』
- 『理性の蝕』
- 『啓蒙の弁証法』

## キーワード（5語）
