# 批判理論

## 概要
批判理論は、社会の支配構造やイデオロギーを批判的に分析し、解放の可能性を探る社会哲学です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| マルクス主義 | 経済基盤分析を文化権力分析へ拡張 | 社会変革志向を共有 |
| ポストモダニズム | 規範理性を保持しつつ相対化を批判 | 権力分析を共有 |
| フェミニズム哲学 | 階級中心とジェンダー中心で焦点差 | 抑圧構造の批判を共有 |
