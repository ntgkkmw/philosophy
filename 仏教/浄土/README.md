# 浄土

## 概要
浄土思想は、阿弥陀仏の本願への信を軸に救済を捉える仏教の系譜です。念仏実践と他力思想を中心に理解を深めます。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 禅 | 信願念仏と坐禅で実践様式が異なる | 日常での倫理実践を重視 |
| 日蓮 | 阿弥陀信仰と法華経中心で経典軸が異なる | 末法意識と救済関心に接点 |
| 原始仏教 | 他力と自力で救済理解が異なる | 苦の克服を目指す点は共通 |
