# 密教

## 概要
密教は、真言・印契・曼荼羅などの象徴体系を通じて即身成仏を説く仏教です。儀礼と宇宙観の統合が特徴です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| チベット仏教 | 地域展開と教団制度が異なる | 真言 印契 曼荼羅の実践体系を共有 |
| 禅 | 象徴儀礼重視と無相実践重視でアプローチが違う | 身体と心の統合修行に接点 |
| 浄土 | 成仏理解の道筋が多様で異なる | 大乗的慈悲と悟り志向は共通 |
