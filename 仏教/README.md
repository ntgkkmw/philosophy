# 仏教

## 概要
仏教は、苦しみの原因を見極め、実践を通じて解脱を目指す思想と実践の体系です。このフォルダでは主要な系譜を比較しながら学びます。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 原始仏教 | 最初期経典に近い実践へ焦点を当てる | 苦の理解と解脱志向を共有 |
| 禅 | 坐禅を軸に直接体験を重視する | 戒定慧の修行枠組みを共有 |
| 浄土 | 他力信心を前面に出す | 救済と慈悲の重視は共通 |
