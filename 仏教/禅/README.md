# 禅

## 概要
禅は、言葉や概念を超えた直接的な覚りの体験を重視する仏教の流れです。坐禅・公案・日常実践を通して心の在り方を鍛えます。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 浄土 | 自己の坐禅実践を重視しやすい | 凡夫救済と慈悲を共有 |
| 密教 | 公案と坐禅を中心に据える | 身体実践を重んじる点で接点 |
| 原始仏教 | 祖師語録と公案の比重が高い | 無常 無我の洞察を共有 |
