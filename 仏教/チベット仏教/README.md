# チベット仏教

## 概要
チベット仏教は、大乗仏教と密教の体系を高度に発展させた伝統です。論理学・瞑想法・菩薩道・タントラ実践を重層的に学びます。

- 基本概念とキーワード
- 代表テキストと重要論点

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 密教 | 学問僧院制度と系譜構造が発達している | タントラ実践と菩薩道を共有 |
| 禅 | 論理学と段階的修道の比重が高い | 瞑想を中核に据える点は共通 |
| 原始仏教 | 大乗密教的展開を強く含む | 縁起と慈悲の理解で接点 |
