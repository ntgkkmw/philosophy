# 原始仏教

## 概要
原始仏教は、釈尊の初期教説に近い層を扱う領域です。四諦・八正道・縁起・無我を中心に、解脱への実践的道筋を確認します。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 禅 | 経典解釈より実践体験を強調しやすい | 無我と縁起の理解を共有 |
| 浄土 | 自力修行中心と他力信仰中心で重心が異なる | 苦からの解放という目標は共通 |
| 密教 | 儀礼体系の多層性よりも基礎教説を重視 | 瞑想と戒律の重要性は接続可能 |
