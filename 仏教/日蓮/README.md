# 日蓮

## 概要
日蓮系の思想は、法華経を中心経典として末法の時代における実践を強調します。題目と社会的実践の結びつきが特徴です。

## このフォルダで扱う観点
- 基本概念とキーワード
- 代表テキストと重要論点
- 他流派・他時代との比較

## 他流派との対比（簡易）
| 比較対象 | 主な違い | 共通点・接点 |
| --- | --- | --- |
| 浄土 | 法華経中心と浄土経典中心で立場が異なる | 在家実践と社会関与の重視に接点 |
| 禅 | 題目実践と坐禅実践で方法が異なる | 末法状況への応答として比較可能 |
| 原始仏教 | 大乗経典を前景化する点が異なる | 苦の超克を目指す宗教実践として接続 |
